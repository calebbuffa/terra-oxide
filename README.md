# terra-oxide

`terra-oxide` is french for "base" or "foundation"contains shared infrastructure for working with 3D geospatial data.

## Workspace

---

- `orkester`: An async runtime and task Runtime.
- .`tairu`: Cesium 3D Tiles
- `zukei`: Geometry\Math
- `moderu`: glTF
- `terra`: Geospatial
- `sovra`: Raster Overlays
- `i3s`: Esri Indexed 3D Scene Layer
- `outil`: Utilities
- `vekuta`: Vector data (points, lines, and polygons)
- `kiban`: 3D Tiles selection
- `coutier`: Asset fetching and loading
- `arazi`: 3D terrain generation and selection
